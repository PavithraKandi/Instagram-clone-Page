# Instagram Clone Page

A simple Instagram Clone built using HTML, CSS, Bootstrap, and Bootstrap Icons. This project recreates the basic Instagram user interface, including stories, posts, image carousel, reels, and navigation sidebar.

## Features

* Instagram-style sidebar navigation
* Stories section with profile circles
* Post feed layout
* Bootstrap image carousel for multiple post images
* Reels/video support
* Like, Comment, Share, and Bookmark icons
* Responsive and modern UI design

## Technologies Used

* HTML5
* CSS3
* Bootstrap 4
* Bootstrap Icons
* Font Awesome

## How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` using Live Server.
4. Explore the Instagram Clone interface.
